## Run Locally

Clone the project

```bash
  git clone https://github.com/Emin14/test-next-js.git
```

Go to the project directory

```bash
  cd test-next-js
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Features

- Getting all posts
- Getting one post
- Receive comments on every post
- Getting the author of the post
- Pagination
- User Login
- User logout
